# MERN Blog Platform

## Features
- Create and publish blog posts
- Admin dashboard
- CI/CD pipeline
- Production monitoring

## Setup
1. Clone repository
2. Configure environment variables
3. Run `npm install` in both folders
4. Start development servers with `npm run dev`
